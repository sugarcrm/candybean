Translations.java is the Java class that translates SodaIceBox tests into another language.

translateTestsSuite is a Bash script that simplifies the process of running the Java program.

See translateTestsSuite for additional documentation.

An example of running translateTestsSuite:
./translateTestsSuite ~/Workspace/Voodoo2/src/main/resources/Translations.properties
